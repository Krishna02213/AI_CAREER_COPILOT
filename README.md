# 🚀 AI Career Copilot

AI Career Copilot is an AI-powered web application that helps users analyze their resumes, identify missing skills for their target job roles, and receive personalized learning recommendations along with interview questions.

## 📌 Features

* 📄 Upload Resume (PDF/DOCX)
* 🤖 AI-powered Resume Analysis
* 🎯 Select your desired career role
* 📊 Skill Gap Analysis
* 📚 Personalized Learning Recommendations
* 💼 Role-specific Interview Questions
* 👤 User Authentication
* 🗂 Resume History Management
* 📱 Responsive User Interface

## ⚙️ How It Works

1. Register or log in.
2. Upload your resume.
3. Choose your target career role.
4. The AI analyzes your resume.
5. View:

   * Existing Skills
   * Missing Skills
   * Learning Roadmap
   * Interview Questions
6. Improve your profile and repeat the analysis anytime.
